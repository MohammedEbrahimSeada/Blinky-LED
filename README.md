### Descirption

An blinking LED project designed with 10 LEDs in shape of flower using a 555 IC. I created this by following the hackclub tutorial at this link: <a href="https://blueprint.hackclub.com/starter-projects/blinky">https://blueprint.hackclub.com/starter-projects/blinky</a>. The board is powered by an external battery with connector in the board.


### Specifications

BOM:
1x NE555 Timer IC (DIP-8)
1x CD4017 Decade Counter IC (DIP-16)
10x LEDs (through-hole)
1x Resistor 1kΩ (R1)
1x Resistor 470Ω (R2)
1x Potentiometer 50kΩ (RV1)
1x Capacitor 1µF (C1)
1x Capacitor 0.01µF (C2)
